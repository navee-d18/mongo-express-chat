# Mini Chat Application

A simple **CRUD Chat Application** built using **Node.js, Express, MongoDB, Mongoose, and EJS**.
This project demonstrates how to build a basic backend application where users can create, view, edit, and delete chat messages.

---

## Features

* Create new chat messages
* View all chats
* Edit existing chat messages
* Delete chats
* MongoDB database integration
* RESTful routing using Express
* Dynamic pages rendered using EJS
* Method Override to support PUT and DELETE requests

---

## Tech Stack

* **Node.js**
* **Express.js**
* **MongoDB**
* **Mongoose**
* **EJS**
* **Method-Override**

##  Installation

Clone the repository

git clone https://github.com/navee-d18/mongo-express-chat.git

Go to project folder

cd mongo-express-chat

Install dependencies

npm install

Start the server

node index.js

---

##  Usage

1. Start MongoDB on your system.
2. Run the server using `node index.js`.
3. Open your browser and go to:

http://localhost:3000/chats

4. Create, edit, and delete chats.
